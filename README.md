# react_styling

## Description

Different ways to style a React app. Scss, css-modules and 2 CSS-in-js libraries.

![screenshot](https://user-images.githubusercontent.com/20776888/115065426-6a6f0d80-9eee-11eb-99bd-650cb3be219d.jpg)

## Installation

To install

`npm install`

To run

`npm start`

## Built With

- React.js
- Scss
- Styled-components
- CSS Modules
- Styled-jsx
